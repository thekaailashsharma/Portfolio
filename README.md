# Kailash Sharma — Builder's Notebook

A personal website designed as an interactive story about a builder. Not a portfolio — a product memo, personal operating system, and builder notebook.

## Architecture

- **Framework**: Vite + React
- **Styling**: Tailwind CSS (CDN)
- **Animation**: Framer Motion
- **Typography**: Instrument Serif, DM Sans, JetBrains Mono

## Key Features

- **Annotation System** — Hover-triggered side notes on key phrases, inspired by editorial margin annotations
- **Interactive Builder Timeline** — Click-to-expand nodes focused on learnings, not job descriptions
- **System Cards** — Problem/solution/impact structure for zero-to-one projects
- **Thought Fragments** — Short reflections on building, ambiguity, and decision making
- **System Notes** — Floating contextual notes beside sections
- **Custom Cursor** — Subtle ambient glow following mouse movement
- **Scroll-driven Reveals** — Staggered fade-in animations on all sections

## Sections

1. Hero — Editorial title with ambient grid background
2. About — Story-driven narrative with annotations and side notes
3. What I Work On — Three expandable pillars (AI, Consumer, Early Execution)
4. Builder Timeline — Interactive learning log
5. Systems I Built — Tripify, Waste2Wealth, Wowdrobe, Evolve with AI
6. Speaking & Mentorship — Events, mentoring philosophy
7. Thinking — Expandable thought fragments
8. Contact — Minimal links section
